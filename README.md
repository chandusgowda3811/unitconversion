Usage
Choose the type of conversion from the dropdown menu.
Select a specific conversion option within the chosen type.
Enter the value to convert in the provided input field.
Click the "Convert" button to see the result.
Supported Conversions
Length
Centimeter to Meter
Meter to Centimeter
Centimeter to Inch
Inch to Centimeter
Mass
Gram to Kilogram
Kilogram to Gram
Temperature
Celsius to Fahrenheit
Fahrenheit to Celsius
Time
Second to Minute
Minute to Second
Speed
Miles per Hour to Kilometers per Hour
Kilometers per Hour to Miles per Hour
Energy
Joule to Kilojoule
Kilojoule to Joule
Pressure
Bar to Pascal
Pascal to Bar
Dependencies
This application uses Python and the tkinter library for the graphical user interface.
